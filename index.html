TIL about the respective roles assigned to HTML, CSS, and Javascript:

  1. HyperText Markup Language, also known as HTML, is used to establish the structure of a web page.
    HTML is a language, as the name suggests, which provides various elements that may appear within a web page.
  2. Cascading Style Sheet, also known as CSS, is used to add aesthetic touch to a web page.
    I think CSS can also be considered a language due to having its' own set of syntaxes. After a style is created, tagging it with corresponding HTML element's tag will lead to the webpage displaying the styled version of said element.
  3. Last but not least, Javascript is used to provide functionalities within a web page.
    I'd argue this is the hardest to master among these 3. On top of the language itself, inadvertent exposure to various frameworks/libraries might cause severe confusion. For beginners, best practice is to master the language itself before getting to frameworks/libraries.
